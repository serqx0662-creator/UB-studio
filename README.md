# UB Studio — портфолио

Многостраничный сайт-портфолио на Next.js 14 (App Router) + TypeScript + Tailwind CSS + Framer Motion.

## Страницы

- `/` — Главная: hero с фото, тизер услуг, избранные проекты, CTA
- `/about` — Обо мне: фото + подход к работе + полный стек технологий
- `/services` — Услуги: 3 тарифных плана с ценами и сроками
- `/portfolio` — Портфолио: карточки всех проектов
- `/portfolio/[slug]` — Отдельная страница проекта: свайп-галерея, полное описание, стек, ссылка на живой сайт
- `/contacts` — Контакты: email/телефон/WhatsApp/GitHub + форма заявки в Telegram

Кнопка **«Связаться»** есть в шапке на каждой странице (десктоп и мобильное меню), в CTA-блоке на главной и на странице контактов.

## Настройка формы обратной связи (Telegram)

Форма на `/contacts` отправляет заявки боту в Telegram. Чтобы это заработало:

1. Напишите **@BotFather** в Telegram → `/newbot` → задайте имя → скопируйте токен вида `123456789:AA...`
2. Напишите своему новому боту любое сообщение (иначе он не сможет писать вам)
3. Откройте в браузере `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates` и найдите `"chat":{"id": ЧИСЛО, ...}` — это ваш `chat_id`
4. Скопируйте `.env.local.example` в `.env.local` и впишите оба значения:

```bash
cp .env.local.example .env.local
```

5. На Vercel эти же переменные нужно добавить в Project Settings → Environment Variables

Без этих переменных форма покажет вежливую ошибку и предложит написать напрямую на почту — сайт не сломается.

## Фото и логотипы проектов

- `public/profile.jpg` — фото на главной и странице «Обо мне»
- `public/projects/*-logo.png` — логотипы проектов (сейчас это сгенерированные заглушки с инициалами — замените на реальные логотипы, когда будут)
- `public/projects/*-1.jpg, *-2.jpg, *-3.jpg` — фото в галерее проекта (тоже заглушки-градиенты — замените на реальные скриншоты интерфейса каждого проекта)

Все пути прописаны в `lib/data.ts`, поле `gallery` и `logo` — просто положите файл с тем же именем в `public/projects/`, ничего в коде менять не нужно.

## Общие элементы (через layout.tsx)

- **Navbar** — навигация по 5 страницам, логотип (`components/Logo.tsx`), кнопка «Связаться», мобильное меню
- **Кастомный курсор** — следует за мышью, увеличивается на интерактивных элементах
- **Анимированный градиентный фон** — теперь заметно ярче, полноценный переход Onyx → Candy Blue
- **Footer**

Все данные (проекты, услуги/цены, навыки, контакты) — в `lib/data.ts` и `lib/countries.ts`.

## Запуск локально

```bash
npm install
cp .env.local.example .env.local   # и вписать токен бота
npm run dev
```

Открыть http://localhost:3000

## Сборка

```bash
npm run build
npm run start
```

## Деплой на Vercel

1. Залей проект в GitHub-репозиторий
2. vercel.com → New Project → выбери репозиторий
3. Добавь `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID` в Environment Variables
4. Deploy — получишь ссылку вида `ub-studio.vercel.app`

## Что можно поменять

- Цвета — `tailwind.config.ts`
- Тексты, проекты, тарифы, контакты — `lib/data.ts`
- Список стран/бюджетов в форме — `lib/countries.ts`
- Шрифты — `app/layout.tsx` (Unbounded / Inter / JetBrains Mono, кириллица)
- Логотип студии — `components/Logo.tsx`
